# Peanut Presence

Show the world what you're building. A clean, lightweight, and reliable Discord Rich Presence extension for VS Code.

## Features

- **Real-time Status**: Automatically updates as you switch files or workspaces.
- **Smart Timer**: Your session clock stays persistent across file changes.
- **Customizable**: Use your own Discord Application IDs and custom assets.
- **Lightweight**: Zero bloat, focus on performance.

## Quick Start

1. **Create an App**: Head to the [Discord Developer Portal](https://discord.com/developers/applications), create a new app, and copy your **Application ID**.
2. **Configure**: Open VS Code settings (`Cmd + ,`), search for **Peanut Presence**, and paste your ID into the `Application ID` field.
3. **Go Live**: Make sure "Display current activity" is enabled in your Discord Activity Privacy settings.

## Commands

Access these via the Command Palette (`Cmd + Shift + P`):
- `Peanut Presence: Enable` — Turn on the presence.
- `Peanut Presence: Disable` — Turn off the presence.
- `Peanut Presence: Reconnect` — Quickly reset the connection.
